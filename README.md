# Samba-Active-Directory-Management-Web-Application
Provide a centralized web interface for IT administrators to manage users, groups, and policies on a Samba-based Active Directory server using a secure, user-friendly platform.

### Featues
 - Non-built-in users - View, Add, Remove, Reset Password, Enabled/Disabled
 - Non-built-in groups - View, Add and Remove
 - Web Filtering - Blocklist
 - User state, logs and activity
 - Client Control (Shutdown or Sign-Off)

### Backend
 - FastAPI - Python 3.7/3.8

### Frontend
- Next.Js - Typescript

### Notes
This is still a work in progress, and contributions/advice/bug reports are welcome.
